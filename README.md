# 基于 GBAE + LightGBM 多算法融合的网络流量异常检测系统

## 项目介绍
针对网络入侵检测场景中的三大核心问题：
1. 数据不平衡
2. 冷启动 / 无标签数据检测
3. 已知攻击与未知攻击无法兼顾

构建了一套**多算法融合智能检测系统**。

## 算法架构
- **LightGBM**：监督学习，负责已知攻击高精度识别
- **GBAE（粒球自编码器）**：无监督学习，负责冷启动与未知攻击泛化
- **模型融合**：加权融合，兼顾精度与泛化能力

## 实验结果
GBAE(无监督-未知攻击) AUC = 0.7860
LightGBM(监督-已知攻击) AUC = 0.9547
融合模型 AUC = 0.9413

## 项目文件
- `cold_start.py` 冷启动检测（问题2）
- `fusion_model.py` 多模型融合（问题3）
- `train_gbae.py` GBAE 模型训练
- `train_lgb.py` LightGBM 训练
- `Utils/data_process.py` 数据预处理
